Frontend developer developes lot of components and integrating all of them might create a havoc/break in total application, so frontend developers test each component/module separetely called unit testing
Angular have testing frameworks like 
   jasmine: popular testing framework default in angular (javascript)
   karma: it launches a test runner into browsers and run the tests, default in angular
   protractor: an end to end testing framework for angular

   There will be a class TestBed which is used in spec files
   for testing you can use "ng test"
